Описание миссии:
Весной 2017 года в провинции Алеппо группа ССО из 20 человек, находясь в непосредственной близости к линии фронта, занималась установлением зданий, удерживаемых противником, опорных пунктов, бронетехники, складов с боеприпасами и маршрутов передвижения. Вся информация передавалась для корректировки авиаударов. Группа была обнаружена и приняла бой против моногочисленной группы боевиков.
