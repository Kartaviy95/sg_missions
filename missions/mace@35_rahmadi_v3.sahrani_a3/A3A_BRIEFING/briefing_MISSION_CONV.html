ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 3000м;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);

УСЛОВНОСТИ ДЛЯ СИНИХ - PMC Avangard (АТАКА)
- Для победы необходимо: освободить Рахмади;
- Регулярним бойцам запрещено использовать парашюты;

УСЛОВНОСТИ ДЛЯ КРАСНИХ - Картель (ОБОРОНА)
- Для победы необходимо: удержать Рахмади;
- Только одному отделению, на выбор КСа обороны, разрешено покидать территорию Рахмади;

УСЛОВИЕ ЗАХВАТА ЗОНЫ:
Для захвата зоны необходимо: присутствие не менее 3 синих и не более 2 красных.

КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКА меньше 3 человек;
Для ОБОРОНА меньше 2 человек;

Ограничение во времени: 40 минут
