ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 2000м;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);

УСЛОВНОСТИ ДЛЯ СИНИХ - Вооруженные силы Украины (ОБОРОНА)
- Для победы необходимо: удержать базудо прибытия подкрепления;
- Всем запрещенно выходить за дорогу отмеченную чёрными точкам;
- Запрещено перекрывать проход на территорию базы;

УСЛОВНОСТИ ДЛЯ КРАСНЫХ - ВС ИРАНА (АТАКА)
- Для победы необходимо: захватить базу;

УСЛОВИЕ ЗАХВАТА ЗОНЫ:
Для захвата зоны необходимо: присутствие не менее 1 синих и 3 красных.

КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКА меньше 3 человек;
Для ОБОРОНА меньше 2 человек;

Ограничение во времени: 30 минут
