A3A_BRIEFING
	*briefing_BLUEFOR_ENEMY - враги синей стороны, вкладка видна только синим
	*briefing_BLUEFOR_TASKS - задачи синей стороны, вкладка видна только синим
	*briefing_MISSC - условности миссии, вкладка видна всем
	*briefing_MISSION_DESCRIPTION - описание миссии, вкладка видна всем
	*briefing_OPFOR_ENEMY - враги красной стороны, вкладка видна только красным
	*briefing_OPFOR_TASKS - задачи красной стороны, вкладка видна только красным

A3A_MODULES - ПРОСТО НЕ ЛЕЗЬТЕ ТУДА, ТАМ РАЗНЫЕ МОДУЛИ АТРИУМА, ОНО ВАМ НЕ НАДО!
A3A_SETUP
	*a3a_header - maxPlayers = ?; выставляйте только вот эту колонку под ваше кол-во слотов, дабы не спамило в РПТ и все. Остальное можно не трогать.
	*a3a_parameters - здесь можно выставить дефолтную для миссии длительность фриз-тайма.
	*infiSTAR_AdminMenu - это не трогаем, это связано с дополнительными античитом;
Equipment - там лежит раздача снаряги по серпу, плюс ваши шаблоны снаряги, да-да придется вручную писать фаилы, в папках оставил примеры. Файлы unit и vehprocessor не трогать.
mis_funcs - не лезем туда, там преинит серп раздачи снаряги.
a3a_client_scripts - клиентские скрипты, если захотите;
a3a_modules - модули а3а, если захотите;
add_med - выдача медицины и беруш;
bn_crew_control - скрипт ограничения посадки в технику, не забудьте в миссии на технику и слоты подвязать переменные!!!
call.fsm - просто не трогайте
description - там можете выставить настройки усложн пилотирования, на данный момент откл., остальное не трогать;
init.sqf - файл инициализации, срабатывает один раз при старте миссии, на данный момент в нем интегрирована инициализация выдачи медицины, прорисовки террейна и надписи в начале игры; тут можно писать скрипты завершения миссии и т п.
initplayerlocal - файл локальной инициализации для игроков, срабатывает при каждом заходе игрока, обработка на машине игрока, тут вписан скрипт выдающий условности при каждом перезаходе, больше ничего не пишите если не знаете как и что.
mission_description - файл описания миссии, внутри все подписано.
mission_parameters - файл настроек миссии, внутри все подписано.
