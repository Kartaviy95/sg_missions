ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 2500;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);
- Медицинской техникой на миссии являются: УАЗы.

УСЛОВНОСТИ ДЛЯ СИНИХ - ЧСО (ОБОРОНА)
- Для победы необходимо: защитить здание правительства;
- Запрещено перекрывать проходы через дыры в бетонном заборе (триггер);
- Всем пехотным отделениям, кроме 1-6, запрещено пересекать красный маркер;

УСЛОВНОСТИ ДЛЯ ЗЕЛЕНЫХ - ЧДКЗ (АТАКА)
- Для победы необходимо: захватить здание правительства;

УСЛОВИЕ ЗАХВАТА ЗОНЫ:
Для захвата зоны необходимо: двукратное преимущество ЧДКЗ в численности внутри здания.

КРИТИЧЕСКИЕ ПОТЕРИ
Для ОБОРОНА меньше 5 человек;
Для АТАКА меньше 6 человек;

Ограничение во времени: 50 минут
