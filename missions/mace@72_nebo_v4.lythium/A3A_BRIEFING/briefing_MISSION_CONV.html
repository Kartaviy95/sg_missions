ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 2000м;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);

УСЛОВНОСТИ ДЛЯ КРАСНЫХ - МП РФ + ССО РФ + ВКС РФ + СпН САА(Атака)
- Для победы необходимо: эвакуировать хотя бы одного пилота в безопасную зону;

УСЛОВНОСТИ ДЛЯ СИНИХ - Боевики(Оборона)
- Для победы необходимо: не дать эвакуировать пилотов.
- Запрещено пересекать и находиться в красном маркере.

УСЛОВНОСТИ ДЛЯ Зеленых - Экипаж Су-24
- Для победы необходимо: эвакуироваться в безопасную зону.
- Точка появления случайна;

КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКИ меньше 7 человек;
Для ОБОРОНЫ меньше 5 человек;

Ограничение во времени: 40 минут
