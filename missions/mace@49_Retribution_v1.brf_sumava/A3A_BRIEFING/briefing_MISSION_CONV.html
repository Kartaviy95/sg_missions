ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 3000м;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);

УСЛОВНОСТИ ДЛЯ Зелёных - Терористы (АТАКА)
- Для победы необходимо: захватить КПП;

УСЛОВНОСТИ ДЛЯ Синих - ДПСУ (ОБОРОНА)
- Для победы необходимо: удержать КПП;
- Запрещено покидать ОТДЕЛЕНИЯ 1-1 КПП(триггер);
- Обороне запрещено виходить за территорию, ограниченную КРАСНЫМИ ТОЧКАМИ!
- Запрещено перекривать дорогу к триггеру!;

УСЛОВИЕ ЗАХВАТА ЗОНЫ:
Для захвата зоны необходимо: присутствие не менее 4 зелёних и не более 3 синих.

КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКА меньше 4 человек;
Для ОБОРОНА меньше 3 человек;

Ограничение во времени: 50 минут
