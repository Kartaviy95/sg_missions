Общие условности:
- Запрещено надевать элементы формы противника(кроме рюкзаков).
- Вся техника может двигаться на фризтайме.
- Все запрещено пересекать границу, обозначенную черным маркером (в т.ч. вертолету). Считать бесконечной.
- Боевой литлберд будет разблокирован через 2 минуты после варбегина.

Технические условности:
- Синие победят, если возьмут под контроль высоту с радиовышкой.. Для этого в зоне триггера должно быть установлено двойное преимущество синих, а красных меньше 4
- Время на миссию - 50 минут. По истечению времени - победа обороны.

- Медицинскую помощь можно получить:
В Хамви с Мк19 - для атаки.
В БТР-80А - для обороны.

- Критические потери:
Для синих - менее 7 человек
Для красных - менее 5 человек
