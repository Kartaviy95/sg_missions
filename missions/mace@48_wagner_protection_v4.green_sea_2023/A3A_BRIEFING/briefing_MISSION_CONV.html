ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 1500м;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);

УСЛОВНОСТИ ДЛЯ СИНИХ - ССНА США (АТАКА)
- Для победы необходимо: эвакуировать агента из тюрьмы (либо уничтожить оборону на её территории);
- Необходимо вывести агента за территорию зелёного эллипса
УСЛОВНОСТИ ДЛЯ КРАСНЫХ - Wagner (ОБОРОНА)
- Для победы необходимо: удержать заложника в тюрьме (либо уничтожить атаку)
- Запрещено выводить заложника за территорию главного здания тюрьмы
- Запрещено стрелять в заложника без причины, разрешено только в случае неподчинения адекватным приказам!
КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКА меньше 4 человек
Для ОБОРОНА меньше 3 человек

УСЛОВНОСТИ ДЛЯ ФИОЛЕТОВЫХ - Agent USA (ГРАЖДАНСКИЙ)
- Для победы необходимо: покинуть территорию ограниченную зелёным эллипсом
- Агенту запрещено копаться в рюкзаках живых игроков!
- Агенту разрешается брать оружие с трупов и стрелять в противника (Вагнеров)
- Слушайся приказов или будешь устранён!

Ограничение во времени: 30 минут
