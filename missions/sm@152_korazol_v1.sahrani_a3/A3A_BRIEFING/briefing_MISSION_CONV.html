ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 1500м;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);
- Медицинской техникой на миссии являются: HMMWV для синих , УАЗ для красных ;

УСЛОВНОСТИ ДЛЯ КРАСНЫХ - SLA (ОБОРОНА)
- Для победы необходимо: удержать многоэтажку;
- Всем запрещено пересекать границу, обозначенную чёрными маркерами;
- Всем, кроме передового взвода(А1-1, А1-2, А1-3), запрещено пересекать границу, обозначенную синими маркерами;

УСЛОВНОСТИ ДЛЯ СИНИХ - RACS (АТАКА)
- Для победы необходимо: захватить многоэтажку;

УСЛОВИЕ ЗАХВАТА ЗОНЫ:
Для захвата зоны необходимо: присутствие не менее 2 синих и полное отсутствие красных.

КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКА меньше 7 человек;
Для ОБОРОНА меньше 5 человек;

Ограничение во времени: 60 минут
