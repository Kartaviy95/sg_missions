Общие условности:
- Запрещено надевать элементы формы противника(кроме рюкзаков).
- Запрещено вести огонь по респаунам сторон в течение двух минут после варбегина.
- Всем запрещено пересекать края карты.

Условности для красных:
- Всем, кроме отделения А1-5, запрещено пересекать границы, обозначенные синими маркерами (внешний забор аэродрома).
- Отделению А1-5 запрещено вывозить за пределы аэродрома любое стационарное вооружение.

Технические условности:
- Синие победят, если уничтожат 4 самолета Су-25.
- Время на миссию - 40 минут. По истечению времени - победа обороны.
- Критические потери для сторон:
Менее 7 синих - победа обороны.
Менее 5 красных - победа атаки.
- Вся техника может двигаться на фризтайме.
- Медицинскую помощь можно получить:
В палатке - для обороны.
Во всех МРАПах - для атаки.
