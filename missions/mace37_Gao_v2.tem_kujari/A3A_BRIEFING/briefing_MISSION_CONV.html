ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 1500m;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);
- Всем запрещено пересекать чёрныё линий (находится можна);

УСЛОВНОСТИ ДЛЯ СИНИХ - ФР (АТАКА)
- Для победы необходимо: захватить Удерживаемый аэропорт;

УСЛОВНОСТИ ДЛЯ КРАСНЫХ - Повстанцев (ОБОРОНА)
- Для победы необходимо: удержать Удерживаемый аэропорт;
- Обороне запрещено покидать территорию, ограниченную чёрной линий (или находится в ней).

УСЛОВИЕ ЗАХВАТА ЗОНЫ:
Для захвата зоны необходимо: присутствие от 2 синих и не более 1 красных игроков.

КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКА 0 человек;
Для ОБОРОНА 0 человек;

Ограничение во времени: 30 минут
