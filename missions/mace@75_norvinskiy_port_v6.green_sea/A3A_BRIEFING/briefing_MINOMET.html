Краткий гайд по использованию миномёта:
1) Необходимо его разложить. Для этого из ящика с миномётом надо достать артиллерийскую плиту и через ACE меню(Ctrl+win) установить её на землю(действие "разложить тренгу"), далее взять на спину ствол миномёта и через ACE взаимодействие(win) соединить две части.
2) Далее, необходимо определить дистанцию стрельбы и азимут цели. Для этого нажмите R на карте по своей позиции и второй раз по точке, куда хотите отправить снаряд. На карте появится линейка с численными значениями, это и есть дистанция, азимут же отображается возле вашей позиции.
3) Следующим шагом будет определение заряда(CH). Для этого возьмите из ящика артиллерийскую таблицу, сядьте на стрелка собранного миномёта и откройте её через ACE меню(Ctrl+win). Слева откроется панель с несколькими значениями от CH-0 до CH-4. Вас интересуют расчёты для осколочных выстрелов. Методом тыка определяете нужный заряд(CH), дальность полёта снаряда с которым соответствует дальности до цели. Второй номер расчёта должен подойти к ящику с БК и через ACE взаимодействие(win) выбрать "подготовить CH-X" по необходимости. Дальше одна из мин в ящике преобразуется в соответствующую. Её надо взять в свой рюкзак и через ACE взаимодействие(win) зарядить в ствол миномёта.
4) Теперь настроим угол(elev). Находим соответствующий столбик в таблице, выбираем значение и с помощью PgUp и PgDn настраиваете угол.
5) Поздравляю, вы великолепны! Можете отправлять 81мм подарок на голову противнику(ну или союзнику).
