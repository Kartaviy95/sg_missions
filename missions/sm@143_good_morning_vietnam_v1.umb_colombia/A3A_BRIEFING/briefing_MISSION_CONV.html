ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 1500м;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);
- Медицинской техникой на миссии являются: UH-1H для синих, мед. палатка для красных ;

УСЛОВНОСТИ ДЛЯ КРАСНЫХ - Вьетконг (ОБОРОНА)
- Для победы необходимо: удержать Лагерь;
- Запрещено пересекать границы, обозначенные синими маркерами;

УСЛОВНОСТИ ДЛЯ СИНИХ - Армия США (АТАКА)
- Для победы необходимо: захватить Лагерь Вьетконга;

УСЛОВИЕ ЗАХВАТА ЗОНЫ:
Для захвата зоны необходимо: присутствие не менее 4 синих и не более 2 красных.

КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКА меньше 8 человек;
Для ОБОРОНА меньше 5 человек;

Ограничение во времени: 60 минут
