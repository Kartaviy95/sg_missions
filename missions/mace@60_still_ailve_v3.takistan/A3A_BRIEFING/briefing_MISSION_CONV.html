ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 2500м;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);

УСЛОВНОСТИ ДЛЯ СИНИХ - SAS (АТАКА)
- Для победы необходимо: захватить Карачинар;

УСЛОВНОСТИ ДЛЯ ЗЕЛЁНЫХ - ИЖЫЛ (ОБОРОНА)
- Для победы необходимо: удержать Карачинар;

УСЛОВИЕ ЗАХВАТА ЗОНЫ:
Для захвата зоны необходимо: 2-х кратное преимущество атакующих в триггере.

КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКА меньше 3 человек;
Для ОБОРОНА меньше 1 человека;

Ограничение во времени: 40 минут
