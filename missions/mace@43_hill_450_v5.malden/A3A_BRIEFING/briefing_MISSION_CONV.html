ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 2500м;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);
- Медицинской техникой на миссии являются: УАЗ 1-1 для красных;
- Всем запрещено находиться в воде ;
- Всем запрещено находиться в Чёрном маркере; Для облегчения ориентирования установлены подбитые БМП-2

УСЛОВНОСТИ ДЛЯ КРАСНЫХ - ЧВК "Восток"
- Для победы необходимо: взять Высоту 450;

УСЛОВНОСТИ ДЛЯ СИНИХ - MAF (Оборона)
- Для победы необходимо: удержать Высоту 450;
- Всем запрещено пересекать границы, обозначенные синими маркерами;

УСЛОВИЕ ЗАХВАТА ЗОНЫ:
Для захвата зоны необходимо: присутствие не менее 2 красных и полное отсутствие синих.

КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКА меньше 3 человек;
Для ОБОРОНА меньше 1 человека;

Ограничение во времени: 30 минут
