ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 2000м;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);
- Медицинской техникой на миссии являются: транспорт 1-1, мед. палатка для красных ;

УСЛОВНОСТИ ДЛЯ КРАСНЫХ - ЧВК "Восток"
- Для победы необходимо: удержать береговую базу;
- Всем запрещено пересекать красный маркер; находиться в нём можно
- Всем запрещено находиться на понтонном мосту;
- Всем запрещено находиться в воде.

УСЛОВНОСТИ ДЛЯ СИНИХ - MSOF (АТАКА)
- Для победы необходимо: захватить Береговую базу;

УСЛОВИЕ ЗАХВАТА ЗОНЫ:
Для захвата зоны необходимо: присутствие не менее 2 синих и полное отсутствие красных.

КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКА меньше 3 человек;
Для ОБОРОНА меньше 2 человек;

Ограничение во времени: 30 минут
