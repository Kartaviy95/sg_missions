ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 2000м;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);
- Медицинской техникой на миссии являются: Для атаки - все автомобили. Для обороны медицинская палатка на стрельбище;

УСЛОВНОСТИ ДЛЯ СИНИХ - 3th SFG + ВС Нигера (ОБОРОНА)
- Для победы необходимо: удержать тренировочную базу в н.п Гуара;
- Всем запрещенно покидать синий елипс!;
- Отделению 1-2 запрещено роумить по карте, их основная задача вернутся в триггер!
- Запрещено перекрывать проход на территорию тренировочной базы (триггер);

УСЛОВНОСТИ ДЛЯ КРАСНЫХ - ИЖЫЛ (АТАКА)
- Для победы необходимо: захватить тренировочную базу в н.п Гуара;

УСЛОВИЕ ЗАХВАТА ЗОНЫ:
Для захвата зоны необходимо: присутствие не менее 2 синих и не более 3 красных.

КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКА меньше 4 человек;
Для ОБОРОНА меньше 3 человек;

Ограничение во времени: 40 минут
