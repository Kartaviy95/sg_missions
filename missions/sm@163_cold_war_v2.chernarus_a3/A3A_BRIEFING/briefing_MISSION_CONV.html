ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 3000м;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);
- Медицинской техникой на миссии являются: БМД-2К, штабные хаммера и палатка в триггере;
- Тепловизоры на технике отключены;

УСЛОВНОСТИ ДЛЯ КРАСНЫХ - ВДВ СССР (АТАКА)
- Для победы необходимо: захватить Зеленогорск;
- Парашюты предназначены только для экипажей вертолетов;
- Для вертолётов есть дополнительный боезапас, на 4 блока С-8КОМ;

УСЛОВНОСТИ ДЛЯ СИНИХ - USMC (ОБОРОНА)
- Для победы необходимо: удержать Зеленогорск;
- Запрещено пересекать дороги отмеченные синими маркерами;

УСЛОВИЕ ЗАХВАТА ЗОНЫ:
Для захвата зоны необходимо: двухкратное преимущество атаки над обороной.

КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКА меньше 6 человек;
Для ОБОРОНА меньше 4 человек;

Ограничение во времени: 80 минут
