ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 2500м;
- ВСЕМ запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);

УСЛОВНОСТИ ДЛЯ КРАСНЫХ - ЧВК "Восток" (АТАКА)
- Для победы необходимо: убить оператора дрона;
- Запрещенно находится в затемненной области;

УСЛОВНОСТИ ДЛЯ СИНИХ - Американское ЧВК (ОБОРОНА)
- Для победы необходимо: защитить оператора дрона;
- Запрещенно выходить/выезжать за синий квадрат!


КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКА меньше 2 человек;
Для ОБОРОНА меньше 2 человека;

Ограничение во времени: 28 минут
