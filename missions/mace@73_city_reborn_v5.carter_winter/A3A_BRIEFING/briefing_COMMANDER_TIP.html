Порядок действий для КСов миссии.
1. Выбрать технику для стороны на брифинге.
1.1. КСу Красной Стороны выбрать следующий сектор для атаки.
1.2. Если выбрали технику для отделений - выбрать респаун рядом с командиром отделения для нее.
1.3. Если выбрали технику в разделе спецтехники - выбрать экипаж для нее.
2. После того как загрузился на фризтайм, если хочешь, можешь взять вооружение со склада через HoldAction. (Вооружение на складе выдается на ВСЮ КАМПАНИЮ, этот склад пополнить в дальнейшем никак не получится)
3. Если ты не выбирал технику для какого-то из своих отделений усиления, то рядом с ними будет коробка, в которой ты можешь выбрать для них бесплатное усиление.
4. Постарайся выиграть миссию для своей стороны :)


Вооружение со склада и усиления в коробках можно выбирать только до конца фризтайма.
Не трать технику стороны бездумно, оставляй ее на последующие отыгрыши для стороны. Техника выдается на ВСЮ КАМПАНИЮ.
