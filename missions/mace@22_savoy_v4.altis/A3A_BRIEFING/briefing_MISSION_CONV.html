ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 1000м;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);

УСЛОВНОСТИ ДЛЯ КРАСНЫХ - ХАМАС (АТАКА)
- Для победы вывести всех живых заложников из гостиницы за синий маркер, либо уничтожить всех боевиков;

УСЛОВНОСТИ ДЛЯ ФИОЛЕТОВЫХ - ГРАЖДАНСКИХ
- Для победы необходимо: покинуть негостепреимный отель синюю область;
- Рекомендуем слушаться пленителей, иначе вы погибните;
- До конца фризтайма запрещено выходить из здания гостиницы (красного маркера);

УСЛОВНОСТИ ДЛЯ ЗЕЛЁНЫХ - Боевики палестинской организации ФАТХ
- Для победы продержаться в здании гостиницы (красный маркер) 20 минут, не дать уйти заложникам;
- Покидать здание гостиницы (красный маркер) ЗАПРЕЩЕНО!;
- Убивать заложников разрешено ТОЛЬКО в случае неподчинения приказу (только адекватные приказы, не просите прыгнуть со здания);

КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКА меньше 3 человек;
Для ОБОРОНА меньше 1 человека;

Ограничение во времени: 20 минут
