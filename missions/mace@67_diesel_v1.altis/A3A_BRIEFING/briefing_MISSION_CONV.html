ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 2000м;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);

УСЛОВНОСТИ ДЛЯ КРАСНЫХ - РУССКИЕ (АТАКА)
- Для победы необходимо: уничтожить склад боеприпасов;

УСЛОВНОСТИ ДЛЯ СИНИХ - АМЕРИКАНЦЫ (ОБОРОНА)
- Для победы необходимо: не дать уничтожить склад боеприпасов;

УСЛОВИЕ УНИЧТОЖЕНИЯ СКЛАДА
- Уничтожить один из ящиков. Ящики можно из автомата, пулемета, ГП-шками, трубой.

КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКА меньше 4 человек;
Для ОБОРОНА меньше 3 человек;

Ограничение во времени: 35 минут
