ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 1500м;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);
- Медицинской техникой на миссии являются: все невооруженные M1240 для синих, мед. палатка для красных ;
- Нельзя заходить за синие маркеры;

УСЛОВНОСТИ ДЛЯ КРАСНЫХ - МП РФ (ОБОРОНА)
- Для победы необходимо: удержать аэропорт;
- Запрещено перекрывать проходы через дыры в заборе;
- Покидать территорию, ограниченную красным маркером, могут только бойцы отделения 1-2;
- Отделению 1-2 запрещно выносить за территорию красного маркера любые РПГ, кроме трофеев.

УСЛОВНОСТИ ДЛЯ СИНИХ - US Army (АТАКА)
- Для победы необходимо: захватить Аэропорт;

УСЛОВИЕ ЗАХВАТА ЗОНЫ:
Для захвата зоны необходимо: присутствие не менее 3 синих и не более 1 красного.

КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКА меньше 4 человека;
Для ОБОРОНА меньше 1 человека;

Ограничение во времени: 30 минут
