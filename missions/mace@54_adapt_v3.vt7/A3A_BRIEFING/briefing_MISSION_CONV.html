ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 3000м;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);

УСЛОВНОСТИ ДЛЯ КРАСНИХ - ВДВ РФ (АТАКА)
- Для победы необходимо: освободить Промышленный завод;

УСЛОВНОСТИ ДЛЯ Зелёных - Ополченцы (ОБОРОНА)
- Для победы необходимо: удержать Промышленный завод;
- Запрещено покидать командиру ополченцов красний (триггер);
- Обороне запрещено покидать территорию, ограниченную синий квадрат

УСЛОВИЕ ЗАХВАТА ЗОНЫ:
Для захвата зоны необходимо: присутствие не менее 4 синих и не более 3 красных.

КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКА меньше 4 человек;
Для ОБОРОНА меньше 3 человек;

Ограничение во времени: 50 минут
