ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 1500м;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);
- На атаку/оборону имеется 30 минут, по истечении времени победа в пользу обороны;
- Условия захвата триггера - больше двух синих, ни одного красного;
- Запрещено открывать двери ангара через Ctrl - пробел;

УСЛОВНОСТИ ДЛЯ КРАСНЫХ
- Запрещено покидать корабль;
- Для победы нужно удержать корабль;

УСЛОВНОСТИ ДЛЯ СИНИХ
- Капитан Прайс лучше всех открывает двери;
- Для победы нужно убить всех террористов на корабле;
- Все вертолёты оборудованы канатами для спуска;
- Спускаясь по канатам на неровную поверхность, вы рискуете покалечиться;
