Крупные силы РФ вышли на край административной границы Чешской республики. Для дальнейшего перехода в наступление русские хотят создать безопасный плацдарм. В глубоких глесах региона Сумавы есть база на которой недавно были размещены американские РСЗО Himars. Если не уничтожить установки, в будущем они могут создать много проблем для русских.
