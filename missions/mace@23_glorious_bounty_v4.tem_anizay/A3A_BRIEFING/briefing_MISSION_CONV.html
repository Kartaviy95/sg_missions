ОБЩИЕ УСЛОВНОСТИ
- Прорисовка: 1500м.
- Запрещено надевать униформу и снаряжение противоположной стороны(рюкзаки можно), снимать очки.
- Ограничение по времени: 20 минут.


УСЛОВНОСТИ ДЛЯ СИНИХ (США, атака)
- Для победы необходимо ликвидировать и обыскать генерала ВСРФ.

УСЛОВНОСТИ ДЛЯ КРАСНЫХ (РФ, оборона)
- Для победы необходимо не дать ликвидировать генерала ВСРФ.
- Всем запрещено выходить за территорию красного эллипса.

КРИТИЧЕСКИЕ ПОТЕРИ
ДЛЯ КРАСНЫХ(ОБОРОНА): менее 2 человек.
ДЛЯ СИНИХ(АТАКА): менее 3 человек.
