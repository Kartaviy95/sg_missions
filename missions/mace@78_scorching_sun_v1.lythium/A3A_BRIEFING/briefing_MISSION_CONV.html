ОБЩИЕ УСЛОВНОСТИ
- Прорисовка - 2500;
- Всем запрещено надевать форму, каски (шлемы и головные уборы), бронежилеты и разгрузки противника (Рюкзаки можно);
- Медицинской техникой на миссии являются: все Humvee, Мед.Hilux;
- Тепловизоры на технике отключены;

УСЛОВНОСТИ ДЛЯ СИНИХ - IDF (АТАКА)
- Для победы необходимо: захватить базу ХАМАС;
- Парашюты предназначены только для экипажей вертолетов;

УСЛОВНОСТИ ДЛЯ КРАСНЫХ - ХАМАС (ОБОРОНА)
- Для победы необходимо: удержать базу;
- Запрещено перекрывать ЛЮБЫЕ проходы внутрь территории базы.(триггер);
- Обороне запрещено покидать территорию, ограниченную красными маркерами.

УСЛОВИЕ ЗАХВАТА ЗОНЫ:
Для захвата зоны необходимо: присутствие не менее 3 синих и не более 2 красных.

КРИТИЧЕСКИЕ ПОТЕРИ
Для АТАКА меньше 5 человек;
Для ОБОРОНА меньше 4 человек;

Ограничение во времени: 50 минут
